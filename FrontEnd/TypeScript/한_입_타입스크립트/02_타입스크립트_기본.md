# 02 원시 타입과 리터럴 타입
## 원시 타입(Primitive Type)
- 하나의 값만 저장하는 타입
- number, string, boolean, null, undefined
## 리터럴 타입(Literal Type)
- 원시 타입의 값을 타입처럼 사용하는 타입
